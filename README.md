# Weather App

A clean and responsive weather application built using **HTML, CSS, and JavaScript**.  
It fetches live weather data using the **OpenWeather API**.

## 🚀 Features

- Search weather by city name
- Live temperature data
- Weather condition and icon
- Humidity, wind speed, feels like, and pressure
- Loading state
- Error handling for invalid city names
- Responsive modern UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeather API
